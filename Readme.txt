# 🐞 Bug Detection and Fixing System using CodeLlama

A multilingual AI-based tool for detecting and fixing bugs in source code written in *Python, Java, and JavaScript. The system analyzes the code, identifies line-wise bugs, and generates corrected code using a fine-tuned CodeLlama model. It features a **FastAPI backend* and an intuitive *Gradio frontend*.

---

## 🚀 Features

- ✅ Supports Python, Java, and JavaScript
- 🤖 CodeLlama-based bug detection (local LLM)
- 📌 Detects multiple types of bugs:
  - Syntax errors
  - Logical errors
  - Runtime errors
- 🧠 Automatically identifies programming language
- 📋 Lists errors with:
  - Line number
  - Error type
  - Explanation
- 🛠 Generates fully corrected code
- 🌐 Web interface via Gradio
- ⚡ FastAPI-powered backend for model inference
- 🪪 Structured output format
- 🔒 Local-only deployment (no API calls)
